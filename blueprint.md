# Apple Game (Fruit Box Clone) - Blueprint

## Overview
이 프로젝트는 인기 있는 '사과게임(Fruit Box)'의 웹 클론 버전입니다. 17x10 그리드에 배치된 사과들의 숫자 합이 10이 되도록 드래그하여 영역을 선택하고, 제한 시간 내에 최대한 많은 사과를 제거하는 게임입니다.

## Project Details
- **기술 스택:** HTML5, Modern CSS (Baseline), Vanilla JavaScript (ES Modules).
- **디자인 스타일:**
  - **Aesthetics:** 생동감 넘치는 색상, 부드러운 애니메이션, 깊이감 있는 그림자 효과.
  - **Interactivity:** 드래그 선택 시 시각적 피드백(하이라이트), 사과 제거 시 애니메이션 효과.
  - **Responsiveness:** 데스크탑과 모바일 기기 모두에서 최적화된 레이아웃.
  - **Typography:** 가독성 좋은 모던한 폰트 (Pretendard).

## Features & Implementation
1. **게임 보드 (17x10 그리드):**
   - 각 셀에는 1부터 9까지의 무작위 숫자가 할당된 사과 아이콘이 배치됩니다.
2. **드래그 선택 로직:**
   - 마우스 또는 터치를 사용하여 직사각형 영역을 드래그하여 선택합니다.
   - 선택된 영역 내의 모든 숫자 합을 실시간으로 계산합니다.
3. **제거 조건:**
   - 선택된 사과들의 숫자 합이 정확히 **10**이면 해당 사과들을 제거합니다.
   - 제거 시 점수를 획득하며, 시각적인 제거 효과를 제공합니다.
4. **점수 및 타이머:**
   - 제거한 사과의 개수에 비례하여 점수를 누적합니다.
   - 기본 120초(2분)의 제한 시간을 제공하며, 시간이 종료되면 게임 오버 화면을 표시합니다.
5. **게임 제어:**
   - 게임 시작, 처음화면으로 이동(다시 시작) 기능을 제공합니다.
   - '다시 시작' 버튼을 누르면 현재 게임을 중단하고 초기 시작 화면으로 돌아갑니다.
